Enhanced Music Player
Features:
- Play / Pause / Next / Previous
- Playlist with durations
- Progress bar and volume control
- Keyboard shortcuts (Space, arrows, + to add)
- Shuffle & Repeat (repeat off/all/one)
- Autoplay toggle
- Drag & drop to add local songs, or press + to open file picker
- Theme toggle (dark/light)
- Album art support (uses assets/* if present)

How to use:
1. Unzip and open index.html in a modern browser.
2. Use controls or keyboard shortcuts to control playback.
3. Drag & drop .mp3/.wav files into the page to add them (they will be loaded via blob URLs).
4. To use your own cover images, add them to assets/ and edit script.js entries for 'cover' paths.
